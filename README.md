# DayOne (static site)

This project is a single-page static HTML site.

## Run locally

From this folder (`c:\Users\rp416\OneDrive\Desktop\DayOne`):

### Option A: Python

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000/`.

### Option B: Node (if you don't have Python)

```powershell
npx --yes serve -l 8000 .
```

Then open `http://localhost:8000/`.
